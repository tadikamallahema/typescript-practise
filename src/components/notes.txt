readonly

--> Makes properties immutable (unable to chaneg value once decalred )

  name: string;
}

Intersection Types:

--> Cover union (|) and intersection (&)
1. Intersection:
type A={name:string;}
type B={age:number;}

type C=A&B; 

const User:C={
name:"Hello",
age:20
};

console.log(User);	//Output :--> { name: 'Hema', age: 20 }

2. Union :

function printId(id: string | number) {
  if (typeof id === "string") {
    console.log(id.toUpperCase());
  } else {
    console.log(id.toFixed(2));
  }
}

printId("15");			--> 15
printId(15) 			-->15.00

Utility Type:
